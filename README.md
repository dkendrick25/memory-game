# Memory game

###Overview of Project

The memory game can be played on 3 different levels, 3 being the hardest. The harder the level is, the more cards there are to match. There is also a moves tracker to show you how well you can do.

##Technologies Used
* HTML5
* CSS3
* JavaScript
* JQuery
